# Spotify Clone

A responsive Spotify web player clone built using HTML, CSS, and JavaScript. This project replicates Spotify’s core user interface and basic playback controls to demonstrate front-end web development skills.

## Features
- Responsive and clean UI mimicking Spotify’s design.
- Play, pause, next, and previous track controls.
- Interactive progress bar to seek within songs.
- Display of currently playing track and album artwork.
- Smooth hover effects and transitions.

## Built With
- HTML5 for structuring the content.
- CSS3 for styling and responsive layouts.
- JavaScript for interactivity and playback control simulation.

## Project Overview
This project focuses on recreating the visual and interactive experience of Spotify’s web player. It does not include backend streaming or authentication but provides a solid front-end foundation that can be extended with APIs for a fully functional music streaming app.

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge).

### Installation
1. Clone the repository:
git clone <your-repo-url>
2. Open `index.html` in your web browser.

### Usage
- Click on play/pause buttons to simulate music playback.
- Use next/previous to switch tracks.
- Drag or click the progress bar to seek in the song.
